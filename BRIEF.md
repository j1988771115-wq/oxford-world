### 🔴 最優先：課程資料

每門課程需要以下資訊：

```
課程名稱：
課程簡介：（2-3 句話）
完整介紹：（一段約 200 字的說明）
講師：
定價：NT$ ___
分類：（商務決策 / 技術實作 / 趨勢分析 / ...）
難度：（入門 / 中級 / 進階）
預估時長：（X 小時）
縮圖：（一張封面圖）
```

**章節結構**（每門課）：
```
章節 1：___（X 分鐘）[免費試看？]
章節 2：___（X 分鐘）
章節 3：___（X 分鐘）
...
```

## 五、技術架構（FYI）

- **前端/後端：** Next.js 16 + React 19
- **資料庫：** Supabase (PostgreSQL) — 新加坡區域
- **Auth：** Supabase Auth（Email + OAuth）
- **AI：** Vercel AI SDK + Claude Sonnet
- **金流：** 藍新金流 NewebPay
- **影片：** Mux（待接通）
- **部署：** Vercel（自動 SSL、CDN）
- **域名：** oxford-vision.com（Cloudflare DNS）
