# 牛津世界官網 — 知識變現平台

## 產品願景

打造「牛津世界」(Oxford World) 官方網站，結合學院品牌展示與知識變現系統，讓使用者能夠瀏覽學院資訊、購買線上課程、訂閱會員、與 AI 智能體互動，並獲取最新報告與影片內容。

## 目標用戶

- 想提升知識與技能的學習者（25-45 歲專業人士）
- 對牛津品牌有興趣的華語圈用戶
- 企業培訓採購決策者

## 核心模組

### 1. 學院介紹 (About)
- 品牌故事、使命願景
- 師資團隊介紹
- 合作夥伴 / 認證

### 2. 線上課程系統 (Courses)
- 課程目錄與分類（商業、科技、領導力等）
- 課程詳情頁（大綱、講師、評價、價格）
- 影片播放器（支援進度追蹤）
- 課程購買 / 解鎖流程
- 學習進度追蹤與完課證書

### 3. 會員系統 (Membership)
- 免費 / 基礎 / 高級三層會員
- 會員專屬內容與折扣
- 訂閱制付款（月繳 / 年繳）
- 會員儀表板（學習進度、收藏、歷史）

### 4. AI 智能體系統 (AI Agents)
- 學習助手：根據課程內容回答問題
- 知識顧問：推薦課程 / 學習路徑
- 互動對話介面（類 ChatGPT）
- 基於課程內容的 RAG 知識庫

### 5. 最新報告與影片 (Insights)
- 研究報告下載（部分免費 / 部分付費）
- 影片內容庫（演講、訪談、教學片段）
- 部落格 / 文章
- 電子報訂閱

### 6. 基礎設施
- 用戶認證（Email + 社群登入）
- 支付整合（Stripe / 藍新金流）
- CMS 後台管理
- SEO 與多語系支援（中/英）
- 響應式設計（Mobile-first）
- 分析與追蹤（GA4 + 自訂事件）

## 技術選型（待定）

- 前端：Next.js 14+ (App Router)
- 後端：Next.js API Routes / tRPC
- 資料庫：PostgreSQL (Supabase / Neon)
- 認證：NextAuth.js / Clerk
- 支付：Stripe
- AI：OpenAI API + LangChain (RAG)
- CMS：Sanity / Strapi
- 部署：Vercel
- 影片：Mux / Cloudflare Stream

## 第一階段目標（MVP）

1. 官網首頁 + 學院介紹
2. 課程目錄 + 課程詳情頁
3. 用戶註冊 / 登入
4. 基礎會員系統（免費 + 付費）
5. 課程購買 + 影片播放
6. 最新報告 / 影片展示頁

## 第二階段

1. AI 智能體系統
2. 學習進度追蹤 + 完課證書
3. 進階會員功能（推薦、收藏）
4. 電子報系統
5. 多語系支援
